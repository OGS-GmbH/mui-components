# Getting started

## Installation

### Prerequisites

- Node.js version 18 or higher.
- A package manager: e.g. npm, pnpm, ...

::: code-group

```sh [npm]
$ npm add @ogs-gmbh/mui-components
```

```sh [pnpm]
$ pnpm add @ogs-gmbh/mui-components
```

```sh [yarn]
$ yarn add @ogs-gmbh/mui-components
```

```sh [bun]
$ bun add @ogs-gmbh/mui-components
```

:::

### Usage

Usage may vary per component, so we recommend to check the [reference](/reference). Every components has an example.
